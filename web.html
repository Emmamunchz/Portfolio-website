02 Grundlæggende web

➜ https://emmamunch.dk/website/


Temaet

I dette tema blev vi introduceret til de mest universelle værktøjer, som er grundlæggende for uddannelsen. Vi arbejdede med centrale begreber inden for design, herunder designprincipper, gestaltlovene, wireframes, prototyper, style tiles, typografi og opsætning af tekst.

Fokus for opgaven var at lære og forstå, hvordan man opbygger et responsivt website, der fungerer på både mobil og computer, ved hjælp af HTML og CSS i VS Code. Opgaven udgjorde samtidig vores studiestartsprøve, hvor vi skulle anvende grid-layout og media queries for at sikre en fleksibel og brugervenlig struktur.


Opgaven

I opgaven “Min computer” lærte jeg, hvordan HTML fungerer som sitets struktur, og hvor vigtigt det er at bygge den korrekt fra starten. Jeg gjorde mig ekstra umage og husker tydeligt, hvor udfordrende det var at forstå grid-layout i begyndelsen. Men gennem opgaven fik jeg styr på det, og nu ser jeg tilbage på den med stolthed. Det er fascinerende at opleve, hvor meget man kan udvikle sig på kort tid.
